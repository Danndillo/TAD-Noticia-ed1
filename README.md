## Rodando Localmente

Clone o projeto

```bash
    git clone https://github.com/Danndillo/TAD-Noticia-ed1
```

Vá para o diretório do projeto

```bash
    cd TAD-Noticia-ed1
```

Compilar o projeto

```bash
    make
```

Execute o projeto
```bash
    ./trabEd1
```
